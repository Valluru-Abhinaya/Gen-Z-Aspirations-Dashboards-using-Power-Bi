# Gen-Z-Aspirations-Dashboards-using-Power-Bi
Developed interactive Power BI dashboards to analyze Gen-Z career preferences and aspirations. Cleaned and transformed data using Excel, created dynamic KPIs with DAX, and designed user-friendly dashboards with filters and visualizations. Delivered insights to support recruitment decisions and HR strategy.

## Data Source

The dataset used in this project was provided by the KultureHire team as part of an internship project. 
Due to privacy and confidentiality requirements, the original dataset cannot be publicly shared.

The analysis and dashboard development were performed using the provided data to generate insights into Gen-Z aspirations, expectations, and career preferences.
